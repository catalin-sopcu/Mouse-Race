This is a Typescript-browser-based mouse race game with real-time scoring, dynamic player interactions, and diverse behaviors for game elements based on their object type, such as evasion, pursuit, and random movement.

-------------------------------------------------------
Use the command "npm install" and "npm start" to start the game.
